Carga de Datos de Entrenamiento

Importa casos de entrenamiento desde archivos CSV o PDF, o añade casos manualmente. Todos los datos se envían al endpoint POST /feedback y quedan listos para re-entrenar el modelo.

Importar desde CSV

El CSV debe tener las columnas titulo, prompt y respuesta. La columna titulo es opcional.

⬇ Descargar plantilla
Formato esperado 
titulo,prompt,respuesta
"Fallo OSPF","El router no propaga rutas OSPF al área backbone","DIAGNÓSTICO: ..."
"DHCP sin IPs","Los clientes no reciben IP...","DIAGNÓSTICO: ..."

Importar desde PDF

Sube un PDF con casos de red. El sistema extrae el texto y lo prepara como casos individuales para que los revises antes de enviar.

ℹ️ El PDF se envía al servidor para procesarlo con pdfplumber. Asegúrate de que el servidor esté activo. El texto extraído se dividirá en casos usando separadores --- o doble salto de línea.
